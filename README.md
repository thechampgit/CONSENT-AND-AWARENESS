# CONSENT-AND-AWARENESS 

Consent & Awareness Hub is a women-centered awareness and support platform designed to help individuals recognize unsafe situations, understand personal boundaries, build confidence in speaking up, and access emotional support resources.

Built as a frontend-focused project using AI-assisted development tools including Antigravity and ChatGPT, the platform combines interactive guidance, educational experiences, and supportive community-driven design into a calm and empathetic digital space.

> ⚠️ Work in Progress — This project is currently under active development.

---

## Vision

The goal of this platform is to create a safe, informative, and empowering digital environment where women can:
- Learn about consent and boundaries
- Prepare for uncomfortable or unsafe situations
- Access emotional guidance
- Feel heard and supported
- Heal through awareness and community

The platform focuses on empathy, preparedness, healing, and empowerment rather than fear-driven awareness.
